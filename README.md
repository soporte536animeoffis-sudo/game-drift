# Styceht Drift Server

Servidor WebSocket para el juego de carreras tipo consola de Anime Offis. Controla el juego desde el celular y juega en navegador.

## Cómo funciona

- El navegador se conecta como receptor
- El celular envía comandos como "acelerar", "frenar", "turbo"
- Railway despliega el servidor y lo conecta entre ambos
